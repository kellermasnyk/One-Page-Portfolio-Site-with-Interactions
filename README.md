# One-Page-Portfolio-Site-with-Interactions
I started with a simple idea; I wanted to vibe code a simple one-page website that initially looked like a simple wall of text. When the user hovered their cursor over certain words, however, pop-ups would display with additional content, like images and videos of portfolio items. Writing is always my jumping-off point when I attempt to solve a problem, so in addition to being engaging, the website would represent my approach to design and problem solving.<br><br>I began with a design of the written content in Figma, then created frames for each of the pop-up windows. Then I linked the project to Figma Make and wrote prompts that specifically mentioned when and where the pop-ups should display. After considerable iteration, I got the windows to display where they wouldn’t obstruct the text. I added a bit more polish on things like the cursor, and called it a project.
